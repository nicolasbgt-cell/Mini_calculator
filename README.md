# Mini Calculator / Mini Calculatrice

---

## 🇬🇧 English

### Description
A mini calculator built in Python with a graphical interface (Tkinter) that performs the 4 basic operations: addition, subtraction, multiplication and division.

### Features
- Addition
- Subtraction
- Multiplication
- Division (with zero division error handling)
- Graphical interface with clickable buttons
- Terminal mode (original version)

### Requirements
- Python 3.10 or higher
- Tkinter (included with Python)

### Installation
```bash
git clone https://github.com/nicolasbgt-cell/Mini_calculator.git
cd Mini_calculator
```

### Usage
```bash
python3 minicalculette.py
```

### Project Structure
```
Mini_calculator/
├── minicalculette.py   # Main code
└── README.md           # Documentation
```

### Description
Une mini calculatrice en Python avec interface graphique (Tkinter) permettant d'effectuer les 4 opérations de base : addition, soustraction, multiplication et division.

### Fonctionnalités
- Addition
- Soustraction
- Multiplication
- Division (avec gestion de la division par zéro)
- Interface graphique avec boutons cliquables
- Mode terminal (version originale)

### Prérequis
- Python 3.10 ou supérieur
- Tkinter (inclus avec Python)

### Installation
```bash
git clone https://github.com/nicolasbgt-cell/Mini_calculator.git
```
